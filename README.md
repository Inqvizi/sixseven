# Six-Seven Gesture Video Player

Цей проєкт використовує комп'ютерний зір (MediaPipe) для розпізнавання жестів рук через веб-камеру. Він створений для відтворення популярного TikTok-тренду "Six Seven". 

Коли програма бачить у кадрі дві відкриті долоні, які рухаються вгору-вниз (ніби терези), вона автоматично вмикає відео з котом (`gato.mp4`) та синхронно відтворює звук (`gato.mp3`). Як тільки рух припиняється або руки зникають з кадру — відео миттєво вимикається.

## Вимоги до системи (Важливо!)
Для стабільної роботи без помилок компіляції (особливо на Windows) настійно рекомендується використовувати **Python версії 3.11 або 3.12**. 
*(Уникайте версій на кшталт 3.13 або 3.14, оскільки багато ML-бібліотек ще не мають під них скомпільованих пакетів).*

Вам також знадобиться підключена та робоча **веб-камера**.

## Структура проєкту
Перед запуском переконайтеся, що всі файли знаходяться в одній папці:

    /Твоя_Папка_Проєкту
    │
    ├── main.py          # Головний скрипт з кодом
    ├── gato.mp4         # Відео, яке буде з'являтися
    ├── gato.mp3         # Аудіодоріжка для відео
    └── README.md        # Цей файл

## Інсталяція та Запуск

**Крок 1. Створіть віртуальне середовище (рекомендовано)**
Відкрийте термінал у папці з проєктом та виконайте:
`python -m venv .venv`

**Крок 2. Активуйте середовище**
* Для Windows (PowerShell/CMD):
  `.\.venv\Scripts\activate`
* Для Mac/Linux:
  `source .venv/bin/activate`

**Крок 3. Встановіть необхідні бібліотеки**
Усі залежності встановлюються однією командою:
`pip install opencv-python mediapipe pygame`

**Крок 4. Запустіть програму**
`python main.py`

## Як користуватися
1. Запустіть скрипт і дочекайтеся, поки увімкнеться веб-камера (з'явиться вікно `Main Camera`).
2. Підніміть **обидві руки** перед камерою так, щоб долоні були відкриті (мінімум 3 випрямлених пальці на кожній руці).
3. Почніть по черзі **рухати руками вгору-вниз**.
4. Програма зафіксує рух і миттєво відкриє вікно `Sixseven` з відео та музикою.
5. Щоб зупинити відео, просто перестаньте махати руками або опустіть їх.

## Як вийти з програми
Щоб повністю закрити програму, **клікніть мишкою на вікно з вашою камерою** (щоб воно стало активним) і натисніть клавішу **Esc** на клавіатурі. Після цього всі вікна закриються, і скрипт завершить роботу.

## Налаштування під себе
У коді є дві змінні, які ви можете змінити для кращої роботи під вашу веб-камеру:
* `MOVEMENT_THRESHOLD = 0.01` — Чутливість до руху. Збільште число, якщо відео вмикається від найменшого тремтіння, або зменште, якщо доводиться махати занадто сильно.
* `MAX_TOLERANCE_FRAMES = 15` — "Запобіжник" від мерехтіння. Кількість кадрів, які програма чекає після зупинки рук, перш ніж вимкнути відео (щоб пробачити мікропаузи під час махання).
